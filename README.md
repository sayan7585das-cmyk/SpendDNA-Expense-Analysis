# 💳 SpendDNA: Personal Expense Analysis Using Python

## 📌 Project Overview

SpendDNA is a data analysis project that explores personal spending behavior using real-world transaction data. The project demonstrates the complete data analysis workflow, including data cleaning, feature engineering, exploratory data analysis (EDA), and interactive visualizations to uncover meaningful spending patterns.

The objective is to transform raw transaction records into actionable insights that help users understand where, when, and how they spend their money.

---

## 🎯 Objectives

* Clean and preprocess transaction data.
* Perform exploratory data analysis (EDA).
* Engineer meaningful features from transaction records.
* Identify major spending categories and merchants.
* Analyze monthly and daily spending trends.
* Generate interactive visualizations for better decision-making.
* Provide insights into personal financial habits.

---

## 📂 Dataset

The dataset contains bank transaction records with attributes such as:

* Transaction Date
* Description
* Amount
* Category (derived)
* Merchant (derived)
* Month
* Day of Week

> **Note:** To protect privacy, sensitive personal information has been removed or anonymized.
